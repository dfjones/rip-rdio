Import a Spotify Collection to Rdio

Log in with Spotify To Get Started

Here's how this works: You will need to export your Rdio collection as a CSV file. I've tested this importer with a CSV file produced by Rdio Enhancer Chrome plugin.

Once you have your CSV file, press the login button above to securely login to your Spotify account using OAuth.

On the next screen, you will see a form where you can upload your CSV file. The importer will then look up all the songs in the file, checking if they exist in Spotify. If a match is found, the song is added to your Spotify collection.

Note: This importer prefers albums over tracks, meaning for each song in your Rdio collection, it will first try to find a matching album in Spotify and, if it is found, this album is added to your Spotify collection. If the album can't be found, it will try to find matching track/artist information and import just the single track. So, if your collection in Rdio consists of a bunch of single tracks, this importer might not work well for you. But the code is on github :-)
